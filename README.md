# roboshield
Linux Robotics Security and Safety Hardening Toolkit

## Features

- Linux security auditing
- SSH hardening
- ROS/ROS2 security checks
- Firewall automation
- USB device monitoring
- Incident reporting

## Installation

```bash
chmod +x install.sh
sudo ./install.sh
```

## Usage

```bash
./scripts/audit.sh
```
